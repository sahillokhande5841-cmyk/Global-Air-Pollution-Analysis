# Global-Air-Pollution-Analysis
Power BI Data Visualization Project on Global Air Pollution Analysis

## Project Overview
This project analyzes global air pollution trends using Power BI dashboards and visualizations.

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Cleaning
- Data Visualization

## Dashboard Features
- AQI Analysis
- Country-wise Pollution
- Region-wise Comparison
- Pollutant Trends
- Interactive Filters and Slicers
